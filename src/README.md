## Project Description
Garage Services is a web application that allows users to book automotive services online. Users can browse available services, select time slots, and confirm appointments instantly. The platform also provides an admin panel to manage services and bookings efficiently.

The application is built using **Java (Servlets & JSP)** for backend logic, **HTML & CSS** for frontend, and **MySQL** for database management.

## Features
- User registration and login
- Browse and select garage services
- View service descriptions and prices
- Real-time slot booking
- Admin panel for managing services and bookings
- Responsive design with glassy UI effects

## Technologies Used
- **Backend:** Java Servlets & JSP
- **Frontend:** HTML5, CSS3
- **Database:** MySQL
- **Web Server:** Apache Tomcat
- **Version Control:** Git, GitHub
